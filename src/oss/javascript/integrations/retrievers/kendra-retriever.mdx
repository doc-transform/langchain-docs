## 概览

[Amazon Kendra](https://aws.amazon.com/kendra/) 是由 Amazon Web Services (AWS) 提供的智能搜索服务。
它利用先进的自然语言处理（NLP）和机器学习算法，为组织内的各种数据源提供强大的搜索功能。
Kendra 旨在帮助用户快速准确地找到所需信息，提高生产力和决策效率。

使用 Kendra，用户可以搜索各种内容类型，包括文档、FAQ、知识库、手册和网站。
它支持多种语言，能够理解复杂查询、同义词和上下文含义，以提供高度相关的搜索结果。

本文将帮助你开始使用 Amazon Kendra [检索器](/oss/langchain/retrieval)。所有 `AWSKendraRetriever` 功能和配置的详细文档请参见 [API 参考](https://api.js.langchain.com/classes/langchain_aws.AmazonKendraRetriever.html)。

## 设置

你需要一个 AWS 账户和一个 Amazon Kendra 实例才能开始使用。参见 AWS 的[本教程](https://docs.aws.amazon.com/kendra/latest/dg/getting-started.html)获取更多信息。

如果你想获取单个查询的自动追踪，你还可以通过取消下方注释来设置你的 [LangSmith](/langsmith/home) API 密钥：

```typescript
// process.env.LANGSMITH_API_KEY = "<YOUR API KEY HERE>";
// process.env.LANGSMITH_TRACING = "true";
```

该检索器位于 `@langchain/aws` 包中：

<CodeGroup>



</CodeGroup>

## 实例化

现在我们可以实例化检索器：

```typescript
import { AmazonKendraRetriever } from "@langchain/aws";

const retriever = new AmazonKendraRetriever({
  topK: 10,
  indexId: "YOUR_INDEX_ID",
  region: "us-east-2", // Your region
  clientOptions: {
    credentials: {
      accessKeyId: "YOUR_ACCESS_KEY_ID",
      secretAccessKey: "YOUR_SECRET_ACCESS_KEY",
    },
  },
});
```

## 用法

```typescript
const query = "..."

await retriever.invoke(query);
```
