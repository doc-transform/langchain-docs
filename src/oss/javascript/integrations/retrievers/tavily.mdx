[Tavily 的 Search API](https://tavily.com) 是一个专为 AI 代理（LLM）构建的搜索引擎，提供实时、准确且事实性的搜索结果。

## 概览

本文将帮助你开始使用 Tavily Search API [检索器](/oss/langchain/retrieval)。所有 `TavilySearchAPIRetriever` 功能和配置的详细文档请参见 [API 参考](https://api.js.langchain.com/classes/langchain_community_retrievers_tavily_search_api.TavilySearchAPIRetriever.html)。

### 集成详情

| Retriever | Source | Package |
| :--- | :--- | :---: |
[`TavilySearchAPIRetriever`](https://api.js.langchain.com/classes/langchain_community_retrievers_tavily_search_api.TavilySearchAPIRetriever.html) | Information on the web. | [`@langchain/community`](https://npmjs.com/@langchain/community/) |

## 设置

你需要将 `TAVILY_API_KEY` 环境变量设置为你的 Tavily API 密钥，或通过构造函数的 `apiKey` 传入。在[官网](https://tavily.com/)注册即可获取密钥。

如果你想获取单个查询的自动追踪，你还可以通过取消下方注释来设置你的 [LangSmith](/langsmith/home) API 密钥：

```typescript
// process.env.LANGSMITH_API_KEY = "<YOUR API KEY HERE>";
// process.env.LANGSMITH_TRACING = "true";
```

### 安装

该检索器位于 `@langchain/community` 包中：

<CodeGroup>
```bash npm
npm install @langchain/community @langchain/core
```
```bash yarn
yarn add @langchain/community @langchain/core
```
```bash pnpm
pnpm add @langchain/community @langchain/core
```
</CodeGroup>

## 实例化

现在我们可以实例化检索器：

```typescript
import { TavilySearchAPIRetriever } from "@langchain/community/retrievers/tavily_search_api";

const retriever = new TavilySearchAPIRetriever({
  k: 3,
});
```

有关允许参数的完整列表，请参见[官方文档](https://docs.tavily.com/docs/tavily-api/rest_api#parameters)。你可以通过 `kwargs` 对象将任何参数传递给 SDK。

## 用法

```typescript
const query = "what is the current weather in SF?";

await retriever.invoke(query);
```
