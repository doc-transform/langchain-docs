---

`ZhipuAIEmbeddings` 类使用智谱AI API 为给定文本生成嵌入。

## 设置

你需要注册智谱AI API 密钥并将其设置为名为 `ZHIPUAI_API_KEY` 的环境变量。

https://open.bigmodel.cn

然后，你需要安装 [`@langchain/community`](https://www.npmjs.com/package/@langchain/community) 包：

<Tip>
See [this section for general instructions on installing LangChain packages](/oss/langchain/install).
</Tip>

```bash npm
npm install @langchain/community @langchain/core jsonwebtoken
```

## 用法

```typescript
import { ZhipuAIEmbeddings } from "@langchain/community/embeddings/zhipuai";

const model = new ZhipuAIEmbeddings({});
const res = await model.embedQuery(
  "What would be a good company name a company that makes colorful socks?"
);
console.log({ res });
```
