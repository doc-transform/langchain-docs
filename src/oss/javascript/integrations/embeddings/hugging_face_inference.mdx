---
title: "HuggingFace 推理集成"
description: "使用 LangChain JavaScript 集成 HuggingFace 推理嵌入模型。"
---

此嵌入集成使用 HuggingFace 推理 API 为给定文本生成嵌入，默认使用 `BAAI/bge-base-en-v1.5` 模型。你可以在构造函数中传入不同的模型名称来使用其他模型。

## 设置

你需要先安装 [`@langchain/community`](https://www.npmjs.com/package/@langchain/community) 包和所需的对等依赖：

<Tip>
See [this section for general instructions on installing LangChain packages](/oss/langchain/install).
</Tip>

```bash npm
npm install @langchain/community @langchain/core @huggingface/inference@4
```
## 用法

```typescript
import { HuggingFaceInferenceEmbeddings } from "@langchain/community/embeddings/hf";

const embeddings = new HuggingFaceInferenceEmbeddings({
  apiKey: "YOUR-API-KEY", // Defaults to process.env.HUGGINGFACEHUB_API_KEY
  model: "MODEL-NAME", // Defaults to `BAAI/bge-base-en-v1.5` if not provided
  provider: "MODEL-PROVIDER", // Falls back to auto selection mechanism within Hugging Face's inference API if not provided
});
```

> **注意：**
> 如果你未提供 `model`，将记录警告并使用默认模型 `BAAI/bge-base-en-v1.5`。
> 如果你未提供 `provider`，Hugging Face 将默认使用 `auto` 选择，这将根据你在 https://hf.co/settings/inference-providers 的设置选择该模型的第一个可用提供商。

> **提示：**
> `hf-inference` 是由 Hugging Face 直接托管的模型的提供商名称。
