## 安装

要访问 DeepSeek 模型，你需要创建一个 DeepSeek 账户、获取 API 密钥，并安装 `@langchain/deepseek` 集成包。

你也可以通过 [Together AI](/oss/integrations/chat/togetherai) 或 [Ollama](/oss/integrations/chat/ollama) 等提供商访问 DeepSeek API。

### 凭据

前往 https://deepseek.com/ 注册 DeepSeek 并生成 API 密钥。完成后设置 `DEEPSEEK_API_KEY` 环境变量：

```bash
export DEEPSEEK_API_KEY="your-api-key"
```

如果你想获取模型调用的自动追踪，你还可以通过取消下方注释来设置你的 [LangSmith](/langsmith/home) API 密钥：

```bash
# export LANGSMITH_TRACING="true"
# export LANGSMITH_API_KEY="your-api-key"
```

### 安装

LangChain ChatDeepSeek 集成位于 `@langchain/deepseek` 包中：

<CodeGroup>
```bash npm
npm install @langchain/deepseek @langchain/core
```
```bash yarn
yarn add @langchain/deepseek @langchain/core
```
```bash pnpm
pnpm add @langchain/deepseek @langchain/core
```
</CodeGroup>

## 实例化

现在我们可以实例化模型对象并生成对话补全：


```typescript
import { ChatDeepSeek } from "@langchain/deepseek";

const llm = new ChatDeepSeek({
  model: "deepseek-reasoner",
  temperature: 0,
  // other params...
})
```


```typescript
const aiMsg = await llm.invoke([
  [
    "system",
    "You are a helpful assistant that translates English to French. Translate the user sentence.",
  ],
  ["human", "I love programming."],
])
aiMsg
```
