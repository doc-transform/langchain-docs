[Amazon Bedrock](https://aws.amazon.com/bedrock/) 是一项完全托管的服务，通过单一 API 提供来自 AI21 Labs、Anthropic、Cohere、Meta、Stability AI 和 Amazon 等领先 AI 公司的高性能基础模型（FM），并提供丰富的能力来帮助你构建具备安全性、隐私保护和负责任 AI 的生成式 AI 应用。

本文将帮助你开始使用 Amazon Bedrock [对话模型](/oss/langchain/models)。所有 `BedrockChat` 功能和配置的详细文档请参见 [API 参考](https://api.js.langchain.com/classes/langchain_community_chat_models_bedrock.BedrockChat.html)。

<Tip>
现已提供更新的 [`ChatBedrockConverse` 对话模型，通过专用的 `@langchain/aws`](/oss/integrations/chat/bedrock_converse) 集成包提供。使用该包可以在更多模型上进行[工具调用](/oss/langchain/tools)。
</Tip>

## 安装

要访问 Bedrock 模型，你需要创建一个 AWS 账户、设置 Bedrock API 服务、获取访问密钥 ID 和秘密密钥，并安装 `@langchain/community` 集成包。

### 凭据

前往 [AWS 文档](https://docs.aws.amazon.com/bedrock/latest/userguide/getting-started.html)注册 AWS 并设置你的凭据。你还需要为你的账户开启模型访问权限，可以[按照这些说明](https://docs.aws.amazon.com/bedrock/latest/userguide/model-access.html)操作。

如果你想获取模型调用的自动追踪，你还可以通过取消下方注释来设置你的 [LangSmith](/langsmith/home) API 密钥：

```bash
# export LANGSMITH_TRACING="true"
# export LANGSMITH_API_KEY="your-api-key"
```

### 安装

LangChain `BedrockChat` 集成位于 `@langchain/community` 包中。你还需要安装几个官方 AWS 包作为对等依赖：

<CodeGroup>
```bash npm
npm install @langchain/community @langchain/core @aws-crypto/sha256-js @aws-sdk/credential-provider-node @smithy/protocol-http @smithy/signature-v4 @smithy/eventstream-codec @smithy/util-utf8 @aws-sdk/types
```
```bash yarn
yarn add @langchain/community @langchain/core @aws-crypto/sha256-js @aws-sdk/credential-provider-node @smithy/protocol-http @smithy/signature-v4 @smithy/eventstream-codec @smithy/util-utf8 @aws-sdk/types
```
```bash pnpm

```
</CodeGroup>

## 实例化

目前，对话模型集成仅支持 Anthropic、Cohere 和 Mistral 模型。对于来自 AI21 或 Amazon 的基础模型，请参见[文本生成 Bedrock 变体](/oss/integrations/llms/bedrock/)。

有几种不同的方式可以向 AWS 进行身份验证——以下示例依赖于在环境变量中设置访问密钥、秘密访问密钥和区域：

```typescript
import { BedrockChat } from "@langchain/community/chat_models/bedrock";

const llm = new BedrockChat({
  model: "anthropic.claude-3-5-sonnet-20240620-v1:0",
  region: process.env.BEDROCK_AWS_REGION,
  credentials: {
    accessKeyId: process.env.BEDROCK_AWS_ACCESS_KEY_ID!,
    secretAccessKey: process.env.BEDROCK_AWS_SECRET_ACCESS_KEY!,
  },
  // endpointUrl: "custom.amazonaws.com",
  // modelKwargs: {
  //   anthropic_version: "bedrock-2023-05-31",
  // },
});
```

Bedrock 模型的工具调用与[其他模型](/oss/langchain/tools)的方式类似，但请注意并非所有 Bedrock 模型都支持工具调用。更多信息请参阅 [AWS 模型文档](https://docs.aws.amazon.com/bedrock/latest/APIReference/welcome.html)。
