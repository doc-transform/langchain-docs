LangChain.js 支持调用 [YandexGPT](https://cloud.yandex.com/en/services/yandexgpt) 对话模型。

## 安装

首先，你需要[创建一个服务账户](https://cloud.yandex.com/en/docs/iam/operations/sa/create)，并赋予 `ai.languageModels.user` 角色。

接下来，你有两种身份验证选择：

- [IAM 令牌](https://cloud.yandex.com/en/docs/iam/operations/iam-token/create-for-sa)。
  你可以在构造函数参数中通过 `iam_token` 指定，或设置环境变量 `YC_IAM_TOKEN`。
- [API 密钥](https://cloud.yandex.com/en/docs/iam/operations/api-key/create)。
  你可以在构造函数参数中通过 `api_key` 指定，或设置环境变量 `YC_API_KEY`。

## 用法

<Tip>
安装 LangChain 包的通用说明请参见[此章节](/oss/langchain/install)。
</Tip>

```bash npm
npm install @langchain/yandex @langchain/core
```

```typescript
import { ChatYandexGPT } from "@langchain/yandex/chat_models";
import { HumanMessage, SystemMessage } from "@langchain/core/messages";

const chat = new ChatYandexGPT();

const res = await chat.invoke([
  new SystemMessage(
    "You are a helpful assistant that translates English to French."
  ),
  new HumanMessage("I love programming."),
]);
console.log(res);

/*
AIMessage {
  lc_serializable: true,
  lc_kwargs: { content: "Je t'aime programmer.", additional_kwargs: {} },
  lc_namespace: [ 'langchain', 'schema' ],
  content: "Je t'aime programmer.",
  name: undefined,
  additional_kwargs: {}
}
 */
```
