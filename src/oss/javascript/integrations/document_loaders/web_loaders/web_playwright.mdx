<Tip>
**兼容性**

仅在 Node.js 上可用。
</Tip>

本示例介绍如何使用 Playwright 从网页加载数据。每个网页创建一个文档。

Playwright 是一个 Node.js 库，提供了用于控制多个浏览器引擎（包括 Chromium、Firefox 和 WebKit）的高级 API。你可以使用 Playwright 自动化网页交互，包括从需要 JavaScript 渲染的动态网页提取数据。

如果你想要更轻量的解决方案，且你要加载的网页不需要 JavaScript 渲染，你可以改用 [`CheerioWebBaseLoader`](/oss/integrations/document_loaders/web_loaders/web_cheerio)。

## 设置

```bash npm
npm install @langchain/community @langchain/core playwright
```
## 用法

```typescript
import { PlaywrightWebBaseLoader } from "@langchain/community/document_loaders/web/playwright";

/**
 * Loader uses `page.content()`
 * as default evaluate function
 **/
const loader = new PlaywrightWebBaseLoader("https://www.tabnews.com.br/");

const docs = await loader.load();
```
## 选项

以下是你可以通过 PlaywrightWebBaseLoaderOptions 接口传递给 PlaywrightWebBaseLoader 构造函数的参数说明：

```typescript
type PlaywrightWebBaseLoaderOptions = {
  launchOptions?: LaunchOptions;
  gotoOptions?: PlaywrightGotoOptions;
  evaluate?: PlaywrightEvaluate;
};
```
1. `launchOptions`：一个可选对象，指定传递给 playwright.chromium.launch() 方法的额外选项。可以包含 headless 标志等选项以在无头模式下启动浏览器。

2. `gotoOptions`：一个可选对象，指定传递给 page.goto() 方法的额外选项。可以包含 timeout 选项来指定最大导航时间（毫秒），或 waitUntil 选项来指定何时视为导航成功。

3. `evaluate`：一个可选函数，可用于使用自定义评估函数在页面上执行 JavaScript 代码。这对于从页面提取数据、与页面元素交互或处理特定 HTTP 响应非常有用。该函数应返回一个解析为包含评估结果字符串的 Promise。

通过将这些选项传递给 `PlaywrightWebBaseLoader` 构造函数，你可以自定义加载器的行为，并使用 Playwright 的强大功能来爬取和交互网页。

以下是一个基本示例：

```typescript
import {
  PlaywrightWebBaseLoader,
  Page,
  Browser,
} from "@langchain/community/document_loaders/web/playwright";

const url = "https://www.tabnews.com.br/";
const loader = new PlaywrightWebBaseLoader(url);
const docs = await loader.load();

// raw HTML page content
const extractedContents = docs[0].pageContent;
```
以下是一个更高级的示例：

```typescript
import {
  PlaywrightWebBaseLoader,
  Page,
  Browser,
} from "@langchain/community/document_loaders/web/playwright";

const loader = new PlaywrightWebBaseLoader("https://www.tabnews.com.br/", {
  launchOptions: {
    headless: true,
  },
  gotoOptions: {
    waitUntil: "domcontentloaded",
  },
  /** Pass custom evaluate, in this case you get page and browser instances */
  async evaluate(page: Page, browser: Browser, response: Response | null) {
    await page.waitForResponse("https://www.tabnews.com.br/va/view");

    const result = await page.evaluate(() => document.body.innerHTML);
    return result;
  },
});

const docs = await loader.load();
```
