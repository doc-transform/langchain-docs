仅在 Node.js 上可用。
</Tip>

本文介绍如何从 S3 文件对象加载文档对象。

## 设置

要运行此示例，你需要已设置好 Unstructured 并在可用的 URL 端点准备就绪。也可以配置为本地运行。

请参阅[此处](/oss/integrations/document_loaders/file_loaders/unstructured)的文档了解如何操作。

你还需要安装官方 AWS SDK：

```bash npm
npm install @langchain/community @langchain/core @aws-sdk/client-s3
```

## 用法

配置好 Unstructured 后，你可以使用 S3 加载器来加载文件并将其转换为文档。

你可以选择提供 s3Config 参数来指定存储桶区域、访问密钥和秘密访问密钥。如果未提供这些参数，你需要在环境中配置它们（例如通过运行 `aws configure`）。

```typescript
import { S3Loader } from "@langchain/community/document_loaders/web/s3";

const loader = new S3Loader({
  bucket: "my-document-bucket-123",
  key: "AccountingOverview.pdf",
  s3Config: {
    region: "us-east-1",
    credentials: {
      accessKeyId: "AKIAIOSFODNN7EXAMPLE",
      secretAccessKey: "wJalrXUtnFEMI/K7MDENG/bPxRfiCYEXAMPLEKEY",
    },
  },
  unstructuredAPIURL: "http://localhost:8000/general/v0/general",
  unstructuredAPIKey: "", // this will be soon required
});

const docs = await loader.load();

console.log(docs);
```
