<Tip>
**兼容性**

仅在 Node.js 上可用。
</Tip>

本文介绍如何使用 [Sonix](https://sonix.ai/) API 从音频文件加载文档对象。

## 设置

要运行此加载器，你需要在 https://sonix.ai/ 创建一个账户并从 https://my.sonix.ai/api 页面获取认证密钥。

你还需要安装 `sonix-speech-recognition` 库：

```bash npm
npm install @langchain/community @langchain/core sonix-speech-recognition
```

## 用法

配置好认证密钥后，你可以使用加载器创建转录并将其转换为文档。
在 `request` 参数中，你可以通过设置 `audioFilePath` 指定本地文件，或使用 `audioUrl` 指定远程文件。
你还需要指定音频语言。请在[此处](https://sonix.ai/docs/api#languages)查看支持的语言列表。

```typescript
import { SonixAudioTranscriptionLoader } from "@langchain/community/document_loaders/web/sonix_audio";

const loader = new SonixAudioTranscriptionLoader({
  sonixAuthKey: "SONIX_AUTH_KEY",
  request: {
    audioFilePath: "LOCAL_AUDIO_FILE_PATH",
    fileName: "FILE_NAME",
    language: "en",
  },
});

const docs = await loader.load();

console.log(docs);
```
