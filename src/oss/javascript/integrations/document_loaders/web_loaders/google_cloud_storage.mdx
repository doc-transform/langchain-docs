<Tip>
**兼容性**

仅在 Node.js 上可用。
</Tip>

本文介绍如何将 Google Cloud Storage 文件加载到 LangChain 文档中。

## 设置

要使用此加载器，你需要已设置好 Unstructured 并在可用的 URL 端点准备就绪。也可以配置为本地运行。

请参阅[此处](/oss/integrations/document_loaders/file_loaders/unstructured)的文档了解如何操作。

你还需要安装官方 Google Cloud Storage SDK：

```bash npm
npm install @langchain/community @langchain/core @google-cloud/storage
```

## 用法

配置好 Unstructured 后，你可以使用 Google Cloud Storage 加载器来加载文件并将其转换为文档。

此外，你可以选择提供 `storageOptions` 参数来指定你的存储选项以及其他认证方式（如果你不想使用应用程序默认凭据（ADC）作为默认方式）。

```typescript
import { GoogleCloudStorageLoader } from "@langchain/community/document_loaders/web/google_cloud_storage";

const loader = new GoogleCloudStorageLoader({
  bucket: "my-bucket-123",
  file: "path/to/file.pdf",
  storageOptions: {
    keyFilename: "/path/to/keyfile.json",
  },
  unstructuredLoaderOptions: {
    apiUrl: "http://localhost:8000/general/v0/general",
    apiKey: "", // this will be soon required
  },
});

const docs = await loader.load();

console.log(docs);
```
