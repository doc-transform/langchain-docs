## 概览

### 集成详情

本示例介绍如何使用 Cheerio 从网页加载数据。每个网页创建一个文档。

Cheerio 是一个快速轻量的库，允许你使用类似 jQuery 的语法解析和遍历 HTML 文档。你可以使用 Cheerio 从网页提取数据，而无需在浏览器中渲染它们。

然而，Cheerio 不模拟网络浏览器，因此无法执行页面上的 JavaScript 代码。这意味着它无法从需要 JavaScript 渲染的动态网页提取数据。要做到这一点，你可以改用 [`PlaywrightWebBaseLoader`](/oss/integrations/document_loaders/web_loaders/web_playwright) 或 [`PuppeteerWebBaseLoader`](/oss/integrations/document_loaders/web_loaders/web_puppeteer)。

## 设置

要访问 `CheerioWebBaseLoader` 文档加载器，你需要安装 `@langchain/community` 集成包和 `cheerio` peer 依赖。

### 凭据

如果你想获得模型调用的自动追踪，你还可以通过取消下方注释来设置你的 [LangSmith](/langsmith/home) API 密钥：

```bash
# export LANGSMITH_TRACING="true"
# export LANGSMITH_API_KEY="your-api-key"
```

### 安装

LangChain CheerioWebBaseLoader 集成位于 `@langchain/community` 包中：

<CodeGroup>
```bash npm
npm install @langchain/community @langchain/core cheerio
```
```bash yarn
yarn add @langchain/community @langchain/core cheerio
```
```bash pnpm
pnpm add @langchain/community @langchain/core cheerio
```
</CodeGroup>

## 实例化

现在我们可以实例化模型对象并加载文档：

```typescript
import { CheerioWebBaseLoader } from "@langchain/community/document_loaders/web/cheerio"

const loader = new CheerioWebBaseLoader("https://news.ycombinator.com/item?id=34817881", {
  // optional params: ...
})
```

## 加载

```typescript
const docs = await loader.load()
docs[0]
```

## 额外配置

`CheerioWebBaseLoader` 在实例化加载器时支持额外配置。以下是如何使用 `selector` 字段的示例，使其只加载提供的 HTML 类名中的内容：

```typescript
import { CheerioWebBaseLoader } from "@langchain/community/document_loaders/web/cheerio"

const loaderWithSelector = new CheerioWebBaseLoader("https://news.ycombinator.com/item?id=34817881", {
  selector: "p",
});

const docsWithSelector = await loaderWithSelector.load();
docsWithSelector[0].pageContent;
```

