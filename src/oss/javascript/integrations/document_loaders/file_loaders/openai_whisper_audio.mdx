**兼容性**

仅在 Node.js 中可用。
</Tip>

本文介绍如何使用 [OpenAI Whisper](https://platform.openai.com/docs/guides/speech-to-text) API 从音频文件加载文档对象。

## 设置

要运行此加载器，你需要在 OpenAI 创建一个账户，并从 https://platform.openai.com/account 页面获取认证密钥。

## 用法

配置好认证密钥后，你可以使用加载器创建转录，然后将其转换为文档。

```typescript
import { OpenAIWhisperAudio } from "@langchain/community/document_loaders/fs/openai_whisper_audio";

const filePath = "./src/document_loaders/example_data/test.mp3";

const loader = new OpenAIWhisperAudio(filePath, {
  transcriptionCreateParams: {
    language: "en",
  },
});

const docs = await loader.load();

console.log(docs);
```
