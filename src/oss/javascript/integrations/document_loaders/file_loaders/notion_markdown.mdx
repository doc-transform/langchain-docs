---
title: "Notion Markdown 导出集成"
description: "使用 LangChain JavaScript 集成 Notion Markdown 导出文档加载器。"
---

本示例介绍如何从 Notion 仪表板导出的 Notion 页面中加载数据。

首先，按照[官方说明](https://www.notion.so/help/export-your-content)将你的 Notion 页面导出为 **Markdown 和 CSV**。确保选择 `包含子页面` 和 `为子页面创建文件夹`。

然后，解压下载的文件并将解压后的文件夹移动到你的仓库中。它应该包含你页面的 Markdown 文件。

文件夹放入仓库后，只需运行以下示例：

```typescript
import { NotionLoader } from "@langchain/community/document_loaders/fs/notion";

export const run = async () => {
  /** Provide the directory path of your notion folder */
  const directoryPath = "Notion_DB";
  const loader = new NotionLoader(directoryPath);
  const docs = await loader.load();
  console.log({ docs });
};
```
