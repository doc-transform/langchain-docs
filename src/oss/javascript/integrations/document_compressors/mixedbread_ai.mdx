## 概览

本指南将帮助你集成和使用 [Mixedbread AI](https://mixedbread.ai/) 重排序 API。重排序 API 允许你根据给定查询对文档列表进行重新排序，从而提升搜索结果或任何排序列表的相关性。

## 安装

要开始使用，请安装 `@langchain/mixedbread-ai` 包：

<Tip>
安装 LangChain 包的通用说明请参见[此章节](/oss/langchain/install)。
</Tip>

```bash
npm install @langchain/mixedbread-ai
```

## 身份验证

通过在 [Mixedbread AI](https://mixedbread.ai/) 注册获取你的 API 密钥。然后你可以设置 `MXBAI_API_KEY` 环境变量，或在构造类时直接作为 `apiKey` 选项传入。

## 使用重排序

`MixedbreadAIReranker` 类提供了对重排序 API 的访问。以下是使用方法：

1. **导入类**：首先，从包中导入 `MixedbreadAIReranker` 类。
```typescript
import { MixedbreadAIReranker } from "@langchain/mixedbread-ai";
```

2. **实例化类**：使用你的 API 密钥创建 `MixedbreadAIReranker` 实例。

```typescript
const reranker = new MixedbreadAIReranker({ apiKey: "your-api-key" });
```

3. **重排序文档**：使用 `rerankDocuments` 方法根据查询对文档重新排序。

```typescript
const documents = [
  { pageContent: "To bake bread you need flour" },
  { pageContent: "To bake bread you need yeast" },
  { pageContent: "To eat bread you need nothing but good taste" },
];
const query = "What do you need to bake bread?";
const result = await reranker.compressDocuments(documents, query);
console.log(result);
```
