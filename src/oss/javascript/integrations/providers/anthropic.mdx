[Anthropic](https://www.anthropic.com/) 是一家 AI 安全与研究公司，是 Claude 的创建者。
本页涵盖了 Anthropic 模型与 LangChain 之间的所有集成。

## 提示词最佳实践

与 OpenAI 模型相比，Anthropic 模型有一些提示词最佳实践。

**系统消息必须是第一条消息**

Anthropic 模型要求所有系统消息必须作为提示词的第一条消息。

## `ChatAnthropic`

`ChatAnthropic` 是 LangChain `ChatModel` 的子类，这意味着它与 `ChatPromptTemplate` 配合使用效果最佳。
你可以通过以下代码导入这个包装器：

<Tip>
安装 LangChain 包的通用说明请参见[此章节](/oss/langchain/install)。
</Tip>

```bash npm
npm install @langchain/anthropic @langchain/core
```
```typescript
import { ChatAnthropic } from "@langchain/anthropic";
const model = new ChatAnthropic({});
```

在使用 ChatModels 时，建议将提示词设计为 `ChatPromptTemplate`。
以下是一个示例：

```typescript
import { ChatPromptTemplate } from "@langchain/classic/prompts";

const prompt = ChatPromptTemplate.fromMessages([
  ["system", "You are a helpful chatbot"],
  ["human", "Tell me a joke about {topic}"],
]);
```

然后可以在链式调用中使用它：

```typescript
const chain = prompt.pipe(model);
await chain.invoke({ topic: "bears" });
```

更多示例（包括多模态输入）请参见[对话模型集成页](/oss/integrations/chat/anthropic/)。
