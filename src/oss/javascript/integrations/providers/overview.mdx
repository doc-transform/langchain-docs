LangChain 集成了各种对话和嵌入模型、工具和工具包、文档加载器、向量存储等服务。

**提供商**是 LangChain 集成的第三方服务或平台，用于访问对话模型、嵌入和向量存储等 AI 能力。这些提供商拥有独立的 `langchain-provider` 包，以更好地进行版本管理、依赖管理和测试。
