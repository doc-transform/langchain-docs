LangChain 通过 `@langchain/google` 包提供了与 [Google AI Studio](https://aistudio.google.com/) 和 [Google Cloud Vertex AI](https://cloud.google.com/vertex-ai) 的集成。

<Info>
  在寻找旧版的 `@langchain/google-genai` 或 `@langchain/google-vertexai` 包？它们仍在[长期支持](#legacy-packages)下维护，但不再推荐用于新项目。
</Info>

## 对话模型

[`ChatGoogle`](/oss/integrations/chat/google) 类是访问 Gemini 模型（如 `gemini-2.5-pro`、`gemini-2.5-flash` 和 `gemini-3.1-pro-preview`）以及 Gemma 等开源模型的推荐方式。它在一个统一接口中同时支持 Google AI Studio 和 Vertex AI。

<Tip>
安装 LangChain 包的通用说明请参见[此章节](/oss/langchain/install)。
</Tip>

<CodeGroup>
```bash npm
npm install @langchain/google @langchain/core
```
```bash yarn
yarn add @langchain/google @langchain/core
```
```bash pnpm
pnpm add @langchain/google @langchain/core
```
</CodeGroup>

配置你的 API 密钥：

```bash
export GOOGLE_API_KEY=your-api-key
```

```typescript
import { ChatGoogle } from "@langchain/google";

const model = new ChatGoogle("gemini-2.5-flash");

const res = await model.invoke([
  ["human", "What would be a good company name for a company that makes colorful socks?"],
]);
```

`ChatGoogle` 支持工具调用、结构化输出、多模态输入（图片、音频、视频）、推理/思考、图片生成、文本转语音，以及 Gemini 原生工具（如 Google 搜索接地和代码执行）。

## 向量存储

### Vertex AI 向量搜索

> [Vertex AI Vector Search](https://cloud.google.com/vertex-ai/docs/matching-engine/overview)（前称 Vertex AI Matching Engine）提供业界领先的大规模低延迟向量数据库。这些向量数据库通常被称为向量相似度匹配或近似最近邻（ANN）服务。

```typescript
import { MatchingEngine } from "@langchain/community/vectorstores/googlevertexai";
```

### Postgres 向量存储

[`@langchain/google-cloud-sql-pg`](https://www.npmjs.com/package/@langchain/google-cloud-sql-pg) 包中的 [PostgresVectorStore](/oss/integrations/vectorstores/google_cloudsql_pg) 模块提供了使用 CloudSQL for PostgreSQL 存储向量嵌入的方法。

```bash
npm install @langchain/google-cloud-sql-pg
```

## 旧版包

以下包在长期支持下维护，供现有用户使用。新项目应使用 `@langchain/google`。

### `@langchain/google-genai`

`@langchain/google-genai` 包提供了 [`ChatGoogleGenerativeAI`](/oss/integrations/chat/google_generative_ai)，用于通过 Google AI Studio 访问 Gemini 模型。该包基于已弃用的 Google SDK 构建，不会再获得新功能。

### `@langchain/google-vertexai`

`@langchain/google-vertexai` 包提供了 [`ChatVertexAI`](/oss/integrations/chat/google_vertex_ai)，用于通过 Vertex AI 访问 Gemini 模型。该包已被 `@langchain/google` 中内置的 Vertex AI 支持取代。

### `@langchain/google-common`

`@langchain/google-common` 包包含 Google 集成包内部使用的共享抽象。它不应被直接使用。

<Tip>
  要从 `@langchain/google-genai` 或 `@langchain/google-vertexai` 迁移到 `@langchain/google`，请参见 [ChatGoogle](/oss/integrations/chat/google) 页面了解配置说明。`ChatGoogle` 类提供等效功能，并统一访问 Google AI Studio 和 Vertex AI。
</Tip>
