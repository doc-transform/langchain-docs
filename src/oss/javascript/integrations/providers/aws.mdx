### SageMaker 端点

> [Amazon SageMaker](https://aws.amazon.com/sagemaker/) 是一个可以通过完全托管的基础设施、工具和工作流来构建、训练和部署机器学习（ML）模型的系统。

我们使用 `SageMaker` 来托管模型并将其作为 `SageMaker Endpoint` 对外提供服务。

参见[使用示例](/oss/integrations/llms/aws_sagemaker)。

```typescript
import {
  SagemakerEndpoint,
  SageMakerLLMContentHandler,
} from "@langchain/community/llms/sagemaker_endpoint";
```

## 文档加载器

### AWS S3 目录和文件

> [Amazon Simple Storage Service (Amazon S3)](https://docs.aws.amazon.com/AmazonS3/latest/userguide/using-folders.html) 是一种对象存储服务。
> [AWS S3 Directory](https://docs.aws.amazon.com/AmazonS3/latest/userguide/using-folders.html) >[AWS S3 Buckets](https://docs.aws.amazon.com/AmazonS3/latest/userguide/UsingBucket.html)

参见 [S3FileLoader 的使用示例](/oss/integrations/document_loaders/web_loaders/s3)。

```bash npm
npm install @aws-sdk/client-s3
```
```typescript
import { S3Loader } from "@langchain/community/document_loaders/web/s3";
```
