[Google Vertex](https://cloud.google.com/vertex-ai) 是一个服务，提供 Google Cloud 中所有可用的基础模型，如 `gemini-2.5-pro`、`gemini-2.5-flash` 等。

本文将帮助你开始使用 VertexAI 文本补全模型（LLM）。`VertexAI` 所有功能和配置选项的详细文档请参见 [API 参考](https://api.js.langchain.com/classes/langchain_google_vertexai.VertexAI.html)。

## 设置

LangChain.js 支持两种不同的身份验证方式，取决于你是在 Node.js 环境还是 Web 环境中运行。

要访问 VertexAI 模型，你需要创建 Google Cloud Platform (GCP) 账户、获取 API 密钥，并安装 `@langchain/google-vertexai` 集成包。

### 凭据

#### Node.js

你需要确保对应项目已启用 Vertex AI API，并且你已通过以下方式之一对 Google Cloud 进行身份验证：

- 你已登录到有权访问该项目的账户（使用 `gcloud auth application-default login`）。
- 你正在使用有权访问该项目的服务账户的机器上运行。
- 你已下载有权访问该项目的服务账户的凭据，并将 `GOOGLE_APPLICATION_CREDENTIALS` 环境变量设置为该文件的路径。
  **或者**
- 你将 `GOOGLE_API_KEY` 环境变量设置为该项目的 API 密钥。

#### Web

要在 Web 环境（如 Edge 函数）中调用 Vertex AI 模型，你需要安装 `@langchain/google-vertexai-web` 包。

然后，你需要将服务账户凭据直接作为 `GOOGLE_VERTEX_AI_WEB_CREDENTIALS` 环境变量添加：

```
GOOGLE_VERTEX_AI_WEB_CREDENTIALS={"type":"service_account","project_id":"YOUR_PROJECT-12345",...}
```

你也可以直接在代码中传递凭据：

```typescript
import { VertexAI } from "@langchain/google-vertexai";
// Or uncomment this line if you're using the web version:
// import { VertexAI } from "@langchain/google-vertexai-web";

const model = new VertexAI({
  authOptions: {
    credentials: {"type":"service_account","project_id":"YOUR_PROJECT-12345",...},
  },
});
```

如果你想获取模型调用的自动追踪，你还可以通过取消下方注释来设置你的 [LangSmith](/langsmith/home) API 密钥：

```bash
# export LANGSMITH_TRACING="true"
# export LANGSMITH_API_KEY="your-api-key"
```

### 安装

LangChain VertexAI 集成位于 `@langchain/google-vertexai` 包中：

<CodeGroup>
```bash npm
npm install @langchain/google-vertexai @langchain/core
```
```bash yarn
yarn add @langchain/google-vertexai @langchain/core
```
```bash pnpm
pnpm add @langchain/google-vertexai @langchain/core
```
</CodeGroup>

或者对于 Web 环境：

<CodeGroup>
```bash npm
npm install @langchain/google-vertexai-web @langchain/core
```
```bash yarn
yarn add @langchain/google-vertexai-web @langchain/core
```
```bash pnpm
pnpm add @langchain/google-vertexai-web @langchain/core
```
</CodeGroup>

## 实例化

现在我们可以实例化模型对象并生成对话补全：

```typescript
import { VertexAI } from "@langchain/google-vertexai-web"

const llm = new VertexAI({
  model: "gemini-pro",
  temperature: 0,
  maxRetries: 2,
  // other params...
})
```

## 调用

```typescript
const inputText = "VertexAI is an AI company that "

const completion = await llm.invoke(inputText)
completion
```

```txt
offers a wide range of cloud computing services and artificial intelligence solutions to businesses and developers worldwide.
```
