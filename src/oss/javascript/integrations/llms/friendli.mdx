> [Friendli](https://friendli.ai/) 通过可扩展、高效的部署方案提升 AI 应用性能并优化成本，专为高负载 AI 工作量量身定制。

本教程将指导你将 `Friendli` 与 LangChain 集成。

## 设置

确保已安装 `@langchain/community`。

<Tip>
安装 LangChain 包的通用说明请参见[此章节](/oss/langchain/install)。
</Tip>

```bash npm
npm install @langchain/community @langchain/core
```

登录 [Friendli Suite](https://suite.friendli.ai/) 创建个人访问令牌，并将其设置为 `FRIENDLI_TOKEN` 环境变量。
你可以将团队 ID 设置为 `FRIENDLI_TEAM` 环境变量。

你可以通过选择想要使用的模型来初始化 Friendli 对话模型。默认模型为 `mixtral-8x7b-instruct-v0-1`。可用模型请参见 [docs.friendli.ai](https://docs.friendli.ai/guides/serverless_endpoints/pricing#text-generation-models)。

## 用法

```typescript
import { Friendli } from "@langchain/community/llms/friendli";

const model = new Friendli({
  model: "mixtral-8x7b-instruct-v0-1", // Default value
  friendliToken: process.env.FRIENDLI_TOKEN,
  friendliTeam: process.env.FRIENDLI_TEAM,
  maxTokens: 18,
  temperature: 0.75,
  topP: 0.25,
  frequencyPenalty: 0,
  stop: [],
});

const response = await model.invoke(
  "Check the Grammar: She dont like to eat vegetables, but she loves fruits."
);

console.log(response);

/*
Correct: She doesn't like to eat vegetables, but she loves fruits
*/

const stream = await model.stream(
  "Check the Grammar: She dont like to eat vegetables, but she loves fruits."
);

for await (const chunk of stream) {
  console.log(chunk);
}

/*
Cor
rect
:
 She
 doesn
...
she
 loves
 fruits
*/
```
