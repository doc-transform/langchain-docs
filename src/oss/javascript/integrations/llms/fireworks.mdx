本文将帮助你开始使用 Fireworks 文本补全模型（LLM）。`Fireworks` 所有功能和配置选项的详细文档请参见 [API 参考](https://api.js.langchain.com/classes/langchain_community_llms_fireworks.Fireworks.html)。

## 设置

要访问 Fireworks 模型，你需要创建 Fireworks 账户、获取 API 密钥，并安装 `@langchain/community` 集成包。

### 凭据

前往 [fireworks.ai](https://fireworks.ai/) 注册 Fireworks 并生成 API 密钥。完成后设置 `FIREWORKS_API_KEY` 环境变量：

```bash
export FIREWORKS_API_KEY="your-api-key"
```

如果你想获取模型调用的自动追踪，你还可以通过取消下方注释来设置你的 [LangSmith](/langsmith/home) API 密钥：

```bash
# export LANGSMITH_TRACING="true"
# export LANGSMITH_API_KEY="your-api-key"
```

### 安装

LangChain Fireworks 集成位于 `@langchain/community` 包中：

<CodeGroup>
```bash npm
npm install @langchain/community @langchain/core
```
```bash yarn
yarn add @langchain/community @langchain/core
```
```bash pnpm
pnpm add @langchain/community @langchain/core
```
</CodeGroup>

## 实例化

现在我们可以实例化模型对象并生成对话补全：

```typescript
import { Fireworks } from "@langchain/community/llms/fireworks"

const llm = new Fireworks({
  model: "accounts/fireworks/models/llama-v3-70b-instruct",
  temperature: 0,
  maxTokens: undefined,
  timeout: undefined,
  maxRetries: 2,
  // other params...
})
```
