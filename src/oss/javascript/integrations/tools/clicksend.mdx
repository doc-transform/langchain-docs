[ClickSend](https://www.clicksend.com/) 是一个基于云的通信平台，使开发者能够通过统一的 API 发送短信、电子邮件、语音、传真和邮件。

## 概览

### 集成详情

| Class | Package | Serializable | [Python support](/oss/integrations/tools/clicksend) | Version |
|:---|:---|:---:|:---:|:---:|
| ClickSendTools | [@clicksend/langchain-clicksend-mcp](https://www.npmjs.com/package/@clicksend/langchain-clicksend-mcp) | ❌ | ❌ | ![npm version](https://img.shields.io/npm/v/@clicksend/langchain-clicksend-mcp?style=flat-square&label=%20) |

### 工具特性

- **短信**：向全球手机号码发送短信
- **电子邮件**：发送事务性和营销电子邮件
- **语音**：发送自动语音消息和电话
- **MCP 集成**：通过模型上下文协议连接，实现无缝代理集成

## 设置

该集成位于 `@clicksend/langchain-clicksend-mcp` 包中。

```bash
npm install @clicksend/langchain-clicksend-mcp
```

### 凭据

你需要一个带 API 凭据的 ClickSend 账户。在 [clicksend.com](https://www.clicksend.com/) 注册以获取你的用户名和 API 密钥。

```typescript Set credentials
import dotenv from 'dotenv';
dotenv.config();
```

## 实例化

通过连接到 ClickSend MCP 服务器来初始化 ClickSend 工具：

```typescript Initialize ClickSend
import { ClickSendTools } from "@clicksend/langchain-clicksend-mcp";

const tools = await ClickSendTools.fromHTTP({
  url: "https://mcp.clicksend.com",
  username: process.env.CLICKSEND_USERNAME,
  apiKey: process.env.CLICKSEND_API_KEY,
});

console.log(`Loaded ${tools.length} ClickSend tools`);
```

## 调用

在你的 LangChain 代理中使用工具发送通信：

```typescript Use with an agent
import { ChatOpenAI } from "@langchain/openai";
import { createAgent } from "langchain";

// Create agent with tools
const agent = await createAgent({
  model: new ChatOpenAI({
    model: "gpt-4o-mini",
    temperature: 0
  }),
  tools,
});

// Use the agent
const result = await agent.invoke({
  messages: [
    { role: "user", content: "Send an SMS to +1234567890 saying 'Hello from LangChain!'" },
  ],
});

console.log(result);
```
