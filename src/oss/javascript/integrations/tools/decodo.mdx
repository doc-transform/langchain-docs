[@decodo/langchain-ts](https://www.npmjs.com/package/@decodo/langchain-ts) 包使开发者能够在 LangChain 应用中使用 Decodo 的网页爬取 API。

网页爬取 API 特性：

- 轻松访问网络数据。简化从网站和在线源检索信息的过程。
- 地理灵活性。无论地域限制如何，都可以访问内容。
- 可靠爬取。先进技术避免被检测和封锁。

## 特性

@decodo/langchain-ts 插件特性：

- **网页爬取**：爬取任意 URL 并获取 Markdown 内容
- **Google 搜索**：搜索 Google 并获取结构化结果
- **Amazon 搜索**：搜索 Amazon 并获取结构化产品数据
- **地理位置选择**，用于位置敏感的内容
- **JavaScript 渲染**，用于需要真实浏览器的目标
- **Markdown 输出**，高效连接 LLM

## 安装

```bash
npm install @decodo/langchain-ts
```

## 快速开始

要使用本项目中的工具，你需要一个 [Decodo 高级网页爬取 API](https://help.decodo.com/docs/web-scraping-api-core-and-advanced-plans) 订阅。可以在[控制台](https://dashboard.decodo.com/)获取免费试用。

## 示例

每个工具的代理示例。

### 通用工具

```typescript
import dotenv from "dotenv";
import { ChatOpenAI } from "@langchain/openai";
import { createAgent } from "@langchain/agents";
import { DecodoUniversalTool } from "@decodo/langchain-ts";

dotenv.config();

const main = async () => {
  const username = process.env.SCRAPER_API_USERNAME!;
  const password = process.env.SCRAPER_API_PASSWORD!;

  const decodoUniversalTool = new DecodoUniversalTool({ username, password });

  const model = new ChatOpenAI({
    model: "gpt-4.1-mini",
  });

  const agent = createAgent({
    llm: model,
    tools: [decodoUniversalTool],
  });

  const result = await agent.invoke({
    messages: [
      {
        role: "user",
        content:
          "scrape the wikipedia NBA 2025 season page and tell me who won in 2025?",
      },
    ],
  });

  console.log(result.messages[result.messages.length - 1].content);
};

if (require.main === module) {
  main();
}
```

更多示例请参见 GitHub 上的 [@decodo/langchain-ts](https://github.com/Decodo/decodo-langchain-ts)。

## 配置

所有工具都接受一个 `DecodoConfig` 对象：

```typescript
type DecodoConfig = {
  username: string; // Your Web Advanced product username
  password: string; // Your Web Advanced product password
};
```
