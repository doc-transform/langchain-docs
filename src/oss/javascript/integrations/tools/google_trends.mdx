Google 趋势工具允许你的代理利用 SerpApi 的 Google Trends API 检索和分析搜索兴趣数据。
这对于了解热门话题、区域搜索兴趣和搜索词的历史流行度非常有用。

API 详情请参见[此处](https://serpapi.com/google-trends-api)

SerpApi 会缓存查询，因此第一次查询会较慢，而后续相同查询会很快。
偶尔相关查询可能无法工作，但随时间变化的兴趣会正常工作。你可以在[此处](https://serpapi.com/playground?engine=google_trends&q=monster&data_type=RELATED_QUERIES)检查你的查询。

## 设置

要使用此工具，你需要配置 SerpApi 的 Google Trends API 访问权限。

从 [SerpApi](https://serpapi.com/users/sign_in) 获取 API 密钥

然后将 API 密钥设置为 `process.env.SERPAPI_API_KEY` 或通过 `apiKey` 构造参数传入。

## 用法

<Tip>
See [this section for general instructions on installing LangChain packages](/oss/langchain/install).
</Tip>

```bash npm
npm install @langchain/openai @langchain/community @langchain/core
```

```typescript
import { SERPGoogleTrendsTool } from "@langchain/community/tools/google_trends";

export async function run() {
  const tool = new SERPGoogleTrendsTool();

  const res = await tool.invoke("Monster");

  console.log(res);
}
```
