本指南将带你创建第一个具备规划、文件系统工具和子 Agent 能力的 Deep Agent。你将构建一个能够进行研究并撰写报告的研究 Agent。

## 前置条件

开始之前，请确保你已拥有模型提供商的 API 密钥（例如 Anthropic、OpenAI）。

<Note>
    Deep Agents 需要支持[工具调用](/oss/langchain/models#tool-calling)的模型。参阅[自定义](/oss/deepagents/customization#model)了解如何配置模型。
</Note>

:::python

</Note>

```bash
export ANTHROPIC_API_KEY="your-api-key"
export TAVILY_API_KEY="your-tavily-api-key"
```

### 步骤 3：创建搜索工具

:::python
```python
import os
from typing import Literal
from tavily import TavilyClient
from deepagents import create_deep_agent

tavily_client = TavilyClient(api_key=os.environ["TAVILY_API_KEY"])

def internet_search(
    query: str,
    max_results: int = 5,
    topic: Literal["general", "news", "finance"] = "general",
    include_raw_content: bool = False,
):
    """Run a web search"""
    return tavily_client.search(
        query,
        max_results=max_results,
        include_raw_content=include_raw_content,
        topic=topic,
    )
```
:::

:::js
```typescript
import { tool } from "langchain";
import { TavilySearch } from "@langchain/tavily";
import { z } from "zod";

const internetSearch = tool(
  async ({
    query,
    maxResults = 5,
    topic = "general",
    includeRawContent = false,
  }: {
    query: string;
    maxResults?: number;
    topic?: "general" | "news" | "finance";
    includeRawContent?: boolean;
  }) => {
    const tavilySearch = new TavilySearch({
      maxResults,
      tavilyApiKey: process.env.TAVILY_API_KEY,
      includeRawContent,
      topic,
    });
    return await tavilySearch._call({ query });
  },
  {
    name: "internet_search",
    description: "运行网络搜索",
    schema: z.object({
      query: z.string().describe("搜索查询词"),
      maxResults: z
        .number()
        .optional()
        .default(5)
        .describe("返回的最大结果数"),
      topic: z
        .enum(["general", "news", "finance"])
        .optional()
        .default("general")
        .describe("搜索主题分类"),
      includeRawContent: z
        .boolean()
        .optional()
        .default(false)
        .describe("是否包含原始内容"),
    }),
  },
);
```
:::

### 步骤 4：创建 Deep Agent

:::python
```python
# 系统提示词引导 Agent 成为专业研究员
research_instructions = """你是一位专业的研究员。你的工作是进行深入研究并撰写一份精彩的报告。

你可以使用网络搜索工具作为收集信息的主要手段。

## `internet_search`

使用此工具对给定查询进行网络搜索。你可以指定返回的最大结果数、主题，以及是否包含原始内容。
"""

agent = create_deep_agent(
    tools=[internet_search],
    system_prompt=research_instructions
)
```
:::

:::js
```typescript
import { createDeepAgent } from "deepagents";

// 系统提示词引导 Agent 成为专业研究员
const researchInstructions = `你是一位专业的研究员。你的工作是进行深入研究并撰写一份精彩的报告。

你可以使用网络搜索工具作为收集信息的主要手段。

## \`internet_search\`

使用此工具对给定查询进行网络搜索。你可以指定返回的最大结果数、主题，以及是否包含原始内容。
`;

const agent = createDeepAgent({
  tools: [internetSearch],
  systemPrompt: researchInstructions,
});
```
:::

### 步骤 5：运行 Agent

:::python
```python
result = agent.invoke({"messages": [{"role": "user", "content": "What is langgraph?"}]})

# 打印 Agent 的响应
print(result["messages"][-1].content)
```
:::

:::js
```typescript
const result = await agent.invoke({
  messages: [{ role: "user", content: "What is langgraph?" }],
});

// 打印 Agent 的响应
console.log(result.messages[result.messages.length - 1].content);
```
:::

## 它是如何工作的？

你的 Deep Agent 会自动执行以下步骤：

1. **规划方案** — 使用内置的 [`write_todos`](/oss/deepagents/harness#to-do-list-tracking) 工具将研究任务分解。
1. **执行研究** — 调用 `internet_search` 工具收集信息。
1. **管理上下文** — 使用文件系统工具（[`write_file`](/oss/deepagents/harness#file-system-access)、[`read_file`](/oss/deepagents/harness#file-system-access)）卸载大量搜索结果。
1. **生成子 Agent** — 根据需要将复杂子任务委派给专业子 Agent。
1. **综合报告** — 将研究结果整合为连贯的响应。

## 流式输出

Deep Agents 内置了[流式输出](/oss/langchain/streaming/overview)支持，可以通过 LangGraph 实时获取 Agent 执行的更新。
这让你能够逐步观察输出，并审查和调试 Agent 及子 Agent 的工作，包括工具调用、工具结果和 LLM 响应。
