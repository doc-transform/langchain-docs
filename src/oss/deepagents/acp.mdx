[Agent 客户端协议 (ACP)](https://agentclientprotocol.com/get-started/introduction) 标准化了编码 Agent 与代码编辑器或 IDE 之间的通信。
借助 ACP 协议，你可以在任何兼容 ACP 的客户端中使用自定义的 Deep Agent，让代码编辑器提供项目上下文并接收丰富的更新信息。

<Note>
    ACP 专为 Agent 与编辑器的集成而设计。如果你需要 Agent 调用外部服务器托管的工具，请参阅 [Model Context Protocol (MCP)](/oss/langchain/mcp/)。
</Note>

## 快速开始

安装 ACP 集成包：

<CodeGroup>
```bash pip
pip install deepagents-acp
```

```bash uv
uv add deepagents-acp
```
</CodeGroup>

然后通过 ACP 暴露一个 Deep Agent。

这会以 stdio 模式启动一个 ACP 服务器（从 stdin 读取请求，将响应写入 stdout）。实际使用中，你通常将其作为 ACP 客户端（例如编辑器）启动的命令来运行，客户端随后通过 stdio 与服务器通信。

:::python

```python icon="server"
import asyncio

from acp import run_agent
from deepagents import create_deep_agent
from langgraph.checkpoint.memory import MemorySaver

from deepagents_acp.server import AgentServerACP


async def main() -> None:
    agent = create_deep_agent(
        # 你可以在此自定义 Deep Agent：设置自定义提示词、
        # 添加自定义工具、附加中间件或组合子 Agent。
        system_prompt="You are a helpful coding assistant",
        checkpointer=MemorySaver(),
    )

    server = AgentServerACP(agent)
    await run_agent(server)


if __name__ == "__main__":
    asyncio.run(main())
```

:::

## 客户端

Deep Agents 可在任何能运行 ACP Agent 服务器的环境中使用。一些知名的 ACP 客户端包括：

- [Zed](https://zed.dev/docs/ai/external-agents)
- [JetBrains IDEs](https://www.jetbrains.com/help/ai-assistant/acp.html)
- Visual Studio Code (via [vscode-acp](https://github.com/formulahendry/vscode-acp))
- Neovim (via ACP-compatible plugins)

### Zed

`deepagents` 仓库包含一个[演示 ACP 入口](https://github.com/langchain-ai/deepagents/blob/main/libs/acp/run_demo_agent.sh)，你可以将其注册到 [Zed](https://zed.dev/docs/ai/external-agents)：

1. 克隆 `deepagents` 仓库并安装依赖：

```bash
git clone https://github.com/langchain-ai/deepagents.git
cd deepagents/libs/acp
uv sync --all-groups
chmod +x run_demo_agent.sh
```

2. 配置演示 Agent 的凭证：

```bash
cp .env.example .env
```

然后在 `.env` 中设置 `ANTHROPIC_API_KEY`。

3. 在 Zed 的 `settings.json` 中配置 ACP Agent 服务器命令：

```json
{
  "agent_servers": {
    "DeepAgents": {
      "type": "custom",
      "command": "/your/absolute/path/to/deepagents/libs/acp/run_demo_agent.sh"
    }
  }
}
```

4. 打开 Zed 的 Agents 面板并启动一个 DeepAgents 线程。

### Toad

如果你想将 ACP Agent 服务器作为本地开发工具运行，可以使用 [Toad](https://github.com/batrachian/toad) 来管理进程。

```bash
uv tool install -U batrachian-toad

toad acp "python path/to/your_server.py" .
# or
toad acp "uv run python path/to/your_server.py" .
```
