---
title: MODEL_AUTHENTICATION
---

<Note>
    目前仅在 `langchainjs`（JavaScript/TypeScript）中使用。
</Note>

你的模型提供商拒绝了你的服务访问请求。

此错误通常发生在身份验证凭据或 API 密钥存在问题时。

## 排查方法

* 确认你的 API 密钥或身份验证凭据正确且有效。
* 如果使用基于环境变量的身份验证，请检查：
    - 变量名拼写是否正确
    - 变量是否已赋值
    - 第三方包（如 `dotenv`）是否干扰了加载过程
* 如果使用代理或非标准端点，请确保你的自定义提供商没有要求替代的身份验证方式。
* 跳过环境变量问题，直接显式传入凭据：

:::python
```python
from langchain_openai import ChatOpenAI

model = ChatOpenAI(api_key="YOUR_KEY_HERE")
```
:::
:::js
```typescript
import { ChatOpenAI } from "@langchain/openai";

const model = new ChatOpenAI({
  apiKey: "YOUR_KEY_HERE",
});
```
:::
