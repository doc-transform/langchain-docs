LangChain 是构建完全自定义 Agent 和 LLM 驱动应用的最便捷方式。
只需不到 10 行代码，即可接入 OpenAI、Anthropic、Google 等[众多模型提供商](/oss/integrations/providers/overview)。
LangChain 提供了预构建的 Agent 架构和模型集成，帮助你快速上手，并将 LLM 无缝融入到 Agent 和应用中。

<Tip>
**LangChain vs. LangGraph vs. Deep Agents**

如果你想构建 Agent，推荐从 [Deep Agents](/oss/deepagents/overview/) 开始。Deep Agents 开箱即用，内置了长对话自动压缩、虚拟文件系统、子 Agent 派生等现代化能力，可有效管理和隔离上下文。

Deep Agents 是基于 LangChain [Agent](/oss/langchain/agents) 的实现。如果你不需要这些能力，或希望为 Agent 和自主应用做更多自定义，可以直接从 LangChain 开始。

当你有更高级的需求，需要将确定性工作流与 Agent 工作流结合、并进行深度定制时，请使用 [LangGraph](/oss/langgraph/overview) —— 我们的底层 Agent 编排框架与运行时。
</Tip>

LangChain 的 [Agent](/oss/langchain/agents) 构建在 LangGraph 之上，从而获得持久执行、流式输出、人机协同、状态持久化等能力。使用 LangChain 的基础 Agent 功能无需了解 LangGraph。

如果你希望快速构建 Agent 和自主应用，推荐使用 LangChain。

## <Icon icon="wand" /> 创建一个 Agent

:::python
```python
# pip install -qU langchain "langchain[anthropic]"
from langchain.agents import create_agent

def get_weather(city: str) -> str:
    """Get weather for a given city."""
    return f"It's always sunny in {city}!"

agent = create_agent(
    model="claude-sonnet-4-6",
    tools=[get_weather],
    system_prompt="You are a helpful assistant",
)

# 运行 Agent
agent.invoke(
    {"messages": [{"role": "user", "content": "what is the weather in sf"}]}
)
```
:::

:::js
```ts
import * as z from "zod";
// npm install @langchain/anthropic to call the model
import { createAgent, tool } from "langchain";

const getWeather = tool(
  ({ city }) => `It's always sunny in ${city}!`,
  {
    name: "get_weather",
    description: "Get the weather for a given city",
    schema: z.object({
      city: z.string(),
    }),
  },
);

const agent = createAgent({
  model: "claude-sonnet-4-6",
  tools: [getWeather],
});

console.log(
  await agent.invoke({
    messages: [{ role: "user", content: "What's the weather in Tokyo?" }],
  })
);
```
:::

参阅[安装指南](/oss/langchain/install)和[快速入门教程](/oss/langchain/quickstart)，开始使用 LangChain 构建你自己的 Agent 和应用。

<Tip>
使用 [LangSmith](/langsmith/home) 追踪请求、调试 Agent 行为并评估输出结果。设置 `LANGSMITH_TRACING=true` 和 API Key 即可开始使用。
</Tip>
