当你准备将 LangChain Agent 部署到生产环境时，LangSmith 提供了专为 Agent 工作负载设计的托管平台。传统托管平台面向无状态、短生命周期的 Web 应用，而 LangGraph 则**专为有状态、长时间运行的 Agent 而构建**，支持持久化状态和后台执行。LangSmith 负责处理基础设施、扩缩容和运维事务，让你可以直接从代码仓库进行部署。

## 前提条件

开始之前，请确保你已具备以下条件：

- 一个 [GitHub 账号](https://github.com/)
- 一个 [LangSmith 账号](https://smith.langchain.com/)（免费注册）

## 部署你的 Agent

### 1. 在 GitHub 上创建仓库

你的应用代码必须托管在 GitHub 仓库中才能部署到 LangSmith。支持公开和私有仓库。在本快速入门中，请先按照[本地服务器配置指南](/oss/langchain/studio#set-up-local-agent-server)确保你的应用兼容 LangGraph，然后将代码推送到仓库。
