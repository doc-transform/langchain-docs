
### 2. 部署到 LangSmith

<Steps>
<Step title="进入 LangSmith 部署页面">
    登录 [LangSmith](https://smith.langchain.com/)。在左侧边栏中选择 **Deployments**。
</Step>
<Step title="创建新部署">
    点击 **+ New Deployment** 按钮。将弹出一个窗格，你可以在其中填写所需字段。
</Step>
<Step title="关联仓库">
    如果你是首次使用或需要添加之前未关联的私有仓库，请点击 **Add new account** 按钮并按指引连接你的 GitHub 账号。
</Step>
<Step title="部署仓库">
    选择你应用的仓库。点击 **Submit** 进行部署。部署可能需要约 15 分钟完成。你可以在 **Deployment details** 视图中查看部署状态。
</Step>
</Steps>

### 3. 在 Studio 中测试你的应用

应用部署完成后：

1. 选择你刚创建的部署以查看更多详情。
2. 点击右上角的 **Studio** 按钮。Studio 将打开并显示你的图。

### 4. 获取部署的 API URL

1. 在 LangGraph 的 **Deployment details** 视图中，点击 **API URL** 将其复制到剪贴板。
2. 点击 `URL` 复制到剪贴板。

### 5. 测试 API

现在你可以测试 API 了：

<Tabs>
:::python
<Tab title="Python">
1. 安装 LangGraph Python：
```shell
pip install langgraph-sdk
```
2. 向 Agent 发送一条消息：
```python
from langgraph_sdk import get_sync_client # or get_client for async

client = get_sync_client(url="your-deployment-url", api_key="your-langsmith-api-key")

for chunk in client.runs.stream(
    None,    # 无线程运行
    "agent", # Agent 名称。在 langgraph.json 中定义。
    input={
        "messages": [{
            "role": "human",
            "content": "What is LangGraph?",
        }],
    },
    stream_mode="updates",
):
    print(f"Receiving new event of type: {chunk.event}...")
    print(chunk.data)
    print("\n\n")
```
</Tab>
:::
:::js
<Tab title="JavaScript">
1. 安装 LangGraph JS：
```shell
npm install @langchain/langgraph-sdk
```
2. 向 Agent 发送一条消息：
```ts
const { Client } = await import("@langchain/langgraph-sdk");

const client = new Client({ apiUrl: "your-deployment-url", apiKey: "your-langsmith-api-key" });

const streamResponse = client.runs.stream(
    null,    // 无线程运行
    "agent", // Agent 名称。在 langgraph.json 中定义。
    {
        input: {
            "messages": [
                { "role": "user", "content": "What is LangGraph?"}
            ]
        },
        streamMode: "messages",
    }
);

for await (const chunk of streamResponse) {
    console.log(`Receiving new event of type: ${chunk.event}...`);
    console.log(JSON.stringify(chunk.data));
    console.log("\n\n");
}
```
</Tab>
:::
